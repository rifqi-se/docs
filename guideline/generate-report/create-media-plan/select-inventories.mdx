---
title: "Select Inventories"
---

To run a campaign, you will, of course, have several **inventories** or **billboards** that will be used as the media to display or run the advertisements for that campaign. To select or add this inventory, you can do so in the **Media Plan - Inventories** menu.

This page will help you understand the **Inventories Tab** and how to add or select the inventory that a campaign's media plan will use to run the creative advertisements.

## Accessing the Media Plan - Inventories

<Steps>
  <Step title="Click the “Inventories” Tab in the Media Plan">
    ![Accessing Media Plan Inventories 1 Pn](/images/AccessingMediaPlan-Inventories-1.png)

    This will display the Inventories Tab.

    ![Accessing Media Plan Inventories 1 2 Pn](/images/AccessingMediaPlan-Inventories-1.2.png)
  </Step>
</Steps>

## Select Inventory in the Media Plan

![Select Inventoryinthe Media Plan Pn](/images/SelectInventoryintheMediaPlan.png)

Based on the image above, here is the explanation for each section and the steps for selecting the inventory to be added to the media plan:

<Steps>
  <Step title="Inventory Category">
    This section functions as a filter to screen the inventory based on its category and display the results in the Inventory List (Point No. 5).

    <img
      src="/images/MemilihInventorydiMediaPlan-12.png"
      alt="Memilih Inventorydi Media Plan 12 Pn"
      title="Memilih Inventorydi Media Plan 12 Pn"
      className="mx-auto"
      style={{ width:"20%" }}
    />
  </Step>
  <Step title="Media Plan Schedule">
    This information indicates the schedule period that will be used as the reference to check whether a billboard slot is **available** during that time range or not.

    <img
      src="/images/MemilihInventorydiMediaPlan-13.png"
      alt="Memilih Inventorydi Media Plan 13 Pn"
      title="Memilih Inventorydi Media Plan 13 Pn"
      className="mx-auto"
      style={{ width:"33%" }}
    />
  </Step>
  <Step title="Search Inventory">
    When you have a large number of inventory or billboards, you can use this search feature to find the inventory you want. You can search for inventory based on the Inventory Name you have specified.

    <img
      src="/images/MemilihInventorydiMediaPlan-14.png"
      alt="Memilih Inventorydi Media Plan 14 Pn"
      title="Memilih Inventorydi Media Plan 14 Pn"
      className="mx-auto"
      style={{ width:"30%" }}
    />
  </Step>
  <Step title="Sort Inventory">
    You can also sort all the inventory you have. The sorting parameters you can use include:

    - **Highest Price**
    - **Lowest Price**
    - **Highest CPM** (Cost Per Mille/Thousand)
    - **Lowest CPM**
    - **Highest Inventory Impression**
    - **Lowest Inventory Impression**

    <img
      src="/images/MemilihInventorydiMediaPlan-15.png"
      alt="Memilih Inventorydi Media Plan 15 Pn"
      title="Memilih Inventorydi Media Plan 15 Pn"
      className="mx-auto"
      style={{ width:"20%" }}
    />
  </Step>
  <Step title="List Inventory">
    This section contains the list of inventory you own and can select to be used in the media plan.

    <img
      src="/images/MemilihInventorydiMediaPlan-16.png"
      alt="Memilih Inventorydi Media Plan 16 Pn"
      title="Memilih Inventorydi Media Plan 16 Pn"
      className="mx-auto"
      style={{ width:"41%" }}
    />
  </Step>
  <Step title="Check Availability Slot">
    Before you can select the inventory, you must first **check the slot availability**. This needs to be done to activate the **'Add Inventory' button**. When an inventory item has available slots, the '**Add Inventory**' button will become active and you can use it to insert the inventory into the **'Selected Inventory' section.**

    <img
      src="/images/MemilihInventorydiMediaPlan-17.png"
      alt="Memilih Inventorydi Media Plan 17 Pn"
      title="Memilih Inventorydi Media Plan 17 Pn"
      className="mx-auto"
      style={{ width:"31%" }}
    />
  </Step>
  <Step title="'Select Inventory' Button">
    To select an inventory item, you can do so by clicking the 'Add' button. After that, an inventory card will appear in the 'Selected Inventory' section.

    <img
      src="/images/MemilihInventorydiMediaPlan-18.png"
      alt="Memilih Inventorydi Media Plan 18 Pn"
      title="Memilih Inventorydi Media Plan 18 Pn"
      className="mx-auto"
      style={{ width:"50%" }}
    />
  </Step>
  <Step title="List Selected Inventory">
    This section contains the list of billboards that you have already selected and can use in the media plan.

    <img
      src="/images/MemilihInventorydiMediaPlan-8.png"
      alt="Memilih Inventorydi Media Plan 8 Pn"
      title="Memilih Inventorydi Media Plan 8 Pn"
      className="mx-auto"
      style={{ width:"50%" }}
    />

    <Note>
      An **inventory card** in the **Selected Inventory** section signifies that you are taking **1 slot** from that inventory item. When there are two cards with the same inventory name, it indicates that you are taking **2 slots** from that inventory item.
    </Note>
  </Step>
  <Step title="'Delete Inventory' Button">
    To delete inventory that you have already selected, you can click the Delete button on the inventory card section.

    <img
      src="/images/MemilihInventorydiMediaPlan-19.png"
      alt="Memilih Inventorydi Media Plan 19 Pn"
      title="Memilih Inventorydi Media Plan 19 Pn"
      className="mx-auto"
      style={{ width:"50%" }}
    />
  </Step>
  <Step title="Selection Inventory Information">
    This section contains a summary of the information regarding the inventory you have selected for a media plan. This information includes:

    - **Total Number of Selected Billboards**
    - **Total Estimated Impressions** generated from all selected billboards
    - **Total CPM** (Cost Per Mille/Thousand) for all selected billboards
    - **Total Price Per Month** for all selected billboards

    <img
      src="/images/MemilihInventorydiMediaPlan-20.png"
      alt="Memilih Inventorydi Media Plan 20 Pn"
      title="Memilih Inventorydi Media Plan 20 Pn"
      className="mx-auto"
      style={{ width:"63%" }}
    />
  </Step>
  <Step title="Click 'Save' Button">
    Once you are confident with the billboards you have selected, you can click the **'Save'** button to proceed.

    The **inventory has been successfully added** to the Media Plan.

    ![Memilih Inventorydi Media Plan 11 2 Pn](/images/MemilihInventorydiMediaPlan-11.2.png)
  </Step>
</Steps>

After adding the inventory, you can perform the actions of generating a campaign report and launching the media plan you have created (See [Launch Media Plan ↗] (/guideline/generate-report/create-media-plan/launch-media-plan)).
