---

Lynx SSP is a Campaign Management System platform designed to simplify the user's workflow in managing campaign planning, creation, inventory management, performance monitoring, and the generation and downloading of campaign reports efficiently.

This page helps you understand the process, from adding inventory and creating campaigns to generating campaign reports, as follows:

![Frame29 Pn](/images/Frame29.png)

  <Step title="Create Inventory on the Lynx SSP Platform" titleSize="p">
    You can first register the inventory you own, whether it's DOOH (Digital-Out-of-Home) or OOH (Out-of-Home) Inventory.

  <Step title="Create Campaign">
    You can fill in information about the campaign you want to run, such as the campaign name, type, and duration.
  </Step>
  <Step title="Create Media Plan">
    After creating a new campaign, you must create a media plan to ensure your campaign runs and meets the client's specifications.
  </Step>
  <Step title="Select Inventory for Media Plan">
    You can select inventories to be included in the media plan, which will be used to run a campaign you have already created.
  </Step>
  <Step title="Generate Report">
    Activate the "generate report" button if the media plan and campaign you're managing require reporting.
  </Step>
  <Step title="Launch Media Plan">
    After completing the media plan creation process, including setting the schedule, selecting inventory, and activating the report generation, you can launch the media plan to run the campaign.
  </Step>
